# ydict

[![Build Status](https://travis-ci.org/ShionRyuu/ydict.svg?branch=master)](https://travis-ci.org/ShionRyuu/ydict)

YouDao Console Version in Go

## Install

```bash
$ go get github.com/ShionRyuu/ydict
```

## Usage

```bash
$ ydict word
```
